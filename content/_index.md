# Create Static Website with Email Integration using AWS Serverless

### Overview

 In this lab, you will learn how to deploy a static website (HTML/CSS) containing a contact form, upload it to AWS S3, connect the form to AWS Lambda and API Gateway, and use Amazon SES to send emails when users fill out the form.
